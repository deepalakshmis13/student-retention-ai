# 🎓 Predictive Student Retention AI System

## 📌 Overview

Student dropouts are often identified too late.
This project presents a **Predictive Student Retention System** that uses **Machine Learning + Data Analytics** to identify at-risk students early and enable proactive intervention.

The system analyzes student data such as **marks, attendance, and engagement** to classify risk levels and provide actionable insights.

---

## 🚀 Features

* 🔍 **Student Risk Prediction** (Low / Medium / High)
* 🤖 **Hybrid AI Model** (Machine Learning + Rule-based)
* 📊 **Interactive Dashboard & Analytics**
* 🧠 **Feature Importance (Explainable AI)**
* 🔎 **Student Search (Name / ID)**
* 📌 **Personalized Recommendations**
* 📩 **Parent Notification Simulation**
* 🎯 **Risk Prediction Simulator**
* 📉 **Intervention Effect Simulation (Novel Feature)**
* 🏆 **Top At-Risk Student Ranking**

---

## 🧠 Machine Learning

* Model: **Random Forest Classifier**
* Features:

  * Marks
  * Attendance
  * Engagement
* Approach:

  * ML-based prediction
  * Rule-based override for extreme cases (Hybrid AI)
* Provides:

  * Real-time predictions
  * Feature importance insights

## 📊 System Workflow

1. Upload student dataset
2. Data is processed and analyzed
3. ML model predicts student risk
4. System generates:

   * Risk classification
   * Recommendations
   * Parent communication
5. Dashboard provides insights for decision-making
